# Los jugadores de Minecraft - Chat

Aplicación de chat en tiempo real con cuentas de usuario, personalización de perfil (imagen o color) y efectos de mouse.

Requisitos
- Node.js 16+

Instalación
1. Clonar el repositorio
2. npm install
3. npm start

Uso
- Abre http://localhost:3000
- Regístrate o inicia sesión
- Entra al chat grupal, personaliza imagen o color de perfil desde tu perfil.

Notas
- Esta es una app de ejemplo. Para producción deberías usar una store de sesiones persistente y un almacenamiento de archivos (S3 u otro).
